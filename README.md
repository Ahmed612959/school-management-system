# 🎓 نظام إدارة معهد رعاية الضبعية الفني للتمريض

نظام متكامل لإدارة المعهد يشمل إدارة الطلاب والنتائج والحضور والاختبارات الإلكترونية.

## ✨ المميزات

- 🔐 **نظام تسجيل دخول** (طلاب - أدمن - ولي أمر)
- 📊 **عرض النتائج** مع رسوم بيانية تفاعلية
- 📅 **نظام الحضور والغياب** مع إحصائيات
- 🤖 **شات بوت ذكي** مع ذاكرة محادثة
- 📝 **نظام اختبارات إلكترونية**
- 👨‍👩‍👧 **لوحة تحكم ولي الأمر** لمتابعة الأبناء
- 🛡️ **حماية متقدمة** للصفحات
- 📱 **تصميم متجاوب** لجميع الأجهزة

## 🛠️ التقنيات المستخدمة

- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Frontend:** HTML5, CSS3, JavaScript
- **AI Integration:** Groq API
- **Charts:** Chart.js
- **Security:** Helmet, CORS, JWT
